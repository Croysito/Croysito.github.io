¿Qué es input()?

input() es una función incorporada de Python: siempre está disponible, sin necesidad de importar nada. Cuando el programa la ejecuta, se detiene y espera a que el usuario escriba algo y presione Enter.

Python
nombre = input("¿Cómo te llamas? ")
print("Hola,", nombre)

Al ejecutar este bloque aparecerá un cuadro pidiendo el nombre. Lo que escribas se guarda en la variable nombre y luego se muestra con print().

Sintaxis básica

La forma más simple es variable = input(). También puedes pasar un mensaje entre paréntesis para indicarle al usuario qué debe escribir.

Python
dato = input()
print("Escribiste:", dato)
Importante

El programa se detiene por completo en la línea de input() hasta que el usuario responde. No se ejecuta nada más hasta ese momento.

El mensaje de solicitud

El texto que colocas entre los paréntesis de input() se llama prompt y se muestra justo antes de esperar la respuesta del usuario. Es totalmente opcional, pero mejora mucho la experiencia del programa.

Python
edad = input("Ingresa tu edad: ")
print("Tu edad es", edad)
Buena práctica

Termina el mensaje con dos puntos y un espacio, por ejemplo "Nombre: ", para que la respuesta del usuario no quede pegada al texto.

input() siempre devuelve texto

Sin importar lo que el usuario escriba, input() siempre devuelve un string. Incluso si tecleas números, Python los recibe como texto.

Python
edad = input("Edad: ")
print(type(edad))

Salida (si escribes 20):

Consola
Edad: 20
<class 'str'>
Error frecuente

Si intentas sumar el resultado de input() directamente con un número, Python producirá un error. Prueba ejecutar este bloque para verlo.

Python
edad = input("Edad: ")
print(edad + 1)

Obtendrás TypeError: can only concatenate str (not "int") to str, porque edad es texto, no un número.

Conversión de tipos

Para trabajar con números, debes convertir el resultado de input() usando int() o float().

Python
edad = int(input("Edad: "))
print(edad + 1)
Python
altura = float(input("Altura en metros: "))
print(altura * 100)

Uso recomendado

Convierte el dato apenas lo recibes, en la misma línea de input(). Así evitas arrastrar un string por el resto del programa cuando en realidad necesitas un número.

Ejemplo práctico

Python
base = float(input("Base del rectángulo: "))
altura = float(input("Altura del rectángulo: "))
area = base * altura
print("El área es:", area)

input() combinado con f-Strings

Una vez leídos y convertidos los datos, los f-Strings son la forma más clara de mostrarlos dentro de un mensaje.

Python
nombre = input("Nombre: ")
edad = int(input("Edad: "))
print(f"Hola {nombre}, el próximo año tendrás {edad + 1} años")
Muy útil

Dentro de las llaves de un f-String puedes usar directamente el resultado convertido, incluso con operaciones como edad + 1.

Leer varios valores en una sola línea

Puedes pedirle al usuario varios datos separados por un mismo input() y luego dividirlos con split().

Python
datos = input("Escribe tu nombre y tu edad separados por coma: ")
nombre, edad = datos.split(",")
print("Nombre:", nombre.strip())
print("Edad:", edad.strip())

También puedes convertir varios números a la vez combinando split() con map().

Python
a, b = map(int, input("Escribe dos números separados por espacio: ").split())
print("Suma:", a + b)
¿Cómo funciona?

split() divide el texto ingresado en una lista de partes, y map(int, ...) convierte cada una de esas partes a número entero.

Validar los datos ingresados

No siempre el usuario escribe lo que el programa espera. Antes de convertir un dato, conviene comprobar que realmente sea válido.

Python
edad = input("Edad: ")
if edad.isdigit():
    edad = int(edad)
    print("Edad registrada:", edad)
else:
    print("Debes ingresar solo números")

Otra forma habitual es usar try y except para capturar el error de conversión directamente.

Python
try:
    edad = int(input("Edad: "))
    print("Correcto:", edad)
except ValueError:
    print("Debes ingresar un número válido")
Uso recomendado

Valida siempre los datos que provienen de input() antes de usarlos en cálculos. Un usuario puede escribir letras donde se esperaba un número.

Errores comunes

❌ Olvidar convertir el tipo

input() siempre devuelve texto. Sumar directamente el resultado con un número produce un TypeError.

❌ Convertir texto no numérico

int("veinte") falla con ValueError. Valida antes de convertir, con isdigit() o try/except.

❌ Olvidar el mensaje

Un input() sin mensaje deja al usuario sin saber qué debe escribir. Siempre incluye un prompt claro.

Prueba el error

Ejecuta este bloque y escribe algo que no sea un número (por ejemplo "veinte") para observar en vivo el ValueError.

Python
edad = int(input("Edad: "))
print("Tu edad es", edad)
Corrígelo

Edita el bloque anterior y envuélvelo en try/except ValueError para mostrar un mensaje amigable en lugar del error.

¿Sabías que...?




🐍 En Python 2 era distinto
Existían raw_input() (devolvía texto) e input() (evaluaba código). Desde Python 3, solo existe input() y siempre devuelve texto.


🚀 input() detiene el programa
Mientras espera una respuesta, ninguna otra línea del programa se ejecuta.


📖 Todo pasa por sys.stdin
Internamente, input() lee desde la entrada estándar del programa.
