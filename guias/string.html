¿Qué es un String?

Un string es una secuencia de caracteres utilizada para almacenar texto. En Python, un string puede contener letras, números, símbolos, espacios y caracteres especiales.

Python
nombre = "Roy"
ciudad = "Cochabamba"
mensaje = "Bienvenido a Python"
print(nombre)
print(ciudad)
print(mensaje)

En Python, un string es un objeto, lo que significa que posee propiedades y métodos que permiten transformarlo fácilmente.

Crear Strings

Existen varias formas de crear una cadena de texto en Python. Todas representan el mismo tipo de dato, pero cada una resulta útil en situaciones diferentes.

Python
nombre = "Roy"
apellido = 'Carrasco'
universidad = "Universidad Adventista de Bolivia"
print(nombre)
print(apellido)
print(universidad)
Importante

Todo lo que esté entre comillas será interpretado por Python como una cadena de texto (str).

Comillas simples y dobles

Python acepta indistintamente comillas simples y dobles.

Python
saludo = "Hola"
mensaje = 'Bienvenido'
print(saludo)
print(mensaje)

Comillas dobles

Son las más utilizadas por la comunidad de Python y la mayoría de ejemplos oficiales.

Comillas simples

Funcionan exactamente igual y suelen emplearse cuando la cadena contiene comillas dobles.

Texto que contiene comillas

Cuando una cadena contiene comillas, resulta conveniente utilizar el otro tipo de comillas para evitar errores.

Python
frase = "Ella dijo 'Hola'"
respuesta = 'Él respondió "Adiós"'
print(frase)
print(respuesta)
Error frecuente

No mezcles el mismo tipo de comillas al inicio y al final de una cadena.

Comillas triples

Las comillas triples permiten escribir texto de varias líneas sin utilizar el carácter \n.

Python
mensaje = """
Bienvenido al sistema.
Este texto ocupa
varias líneas.
"""
print(mensaje)

También se utilizan para escribir documentación dentro de funciones y clases.

String vacío

Una cadena también puede estar vacía. Esto es muy útil cuando una variable almacenará texto más adelante.

Python
nombre = ""
apellido = ''
print(nombre)
print(apellido)
Recuerda

Una cadena vacía tiene longitud cero, pero sigue siendo un objeto de tipo str.

Acceder a los caracteres (Índices)

Cada carácter de un string posee una posición llamada índice. Los índices comienzan siempre en 0.

Idea importante

Python numera cada carácter individualmente, incluyendo espacios.

012345
Python
Python
texto = "Python"
print(texto[0])
print(texto[1])
print(texto[5])

texto[0]

Devuelve la primera letra.

texto[1]

Devuelve la segunda letra.

texto[5]

Devuelve la última letra.

Índices negativos

Python también puede contar desde el final utilizando números negativos.

-6-5-4-3-2-1

Python

Muy útil

Los índices negativos permiten acceder rápidamente al último elemento sin conocer la longitud del string.

Ejemplos prácticos

Los espacios también cuentan

Un error frecuente es pensar que los espacios no tienen posición.

Python
frase = "Hola Mundo"
print(frase[4])
print(frase[5])
Resultado

La posición 4 corresponde a la letra a, mientras que la posición 5 corresponde al espacio entre las palabras.

Error común: IndexError

Si intentas acceder a una posición que no existe, Python producirá un error. Prueba ejecutar este bloque para ver el error real.

Python
texto = "Python"
print(texto[20])
¿Por qué ocurre?

La palabra Python solamente posee seis caracteres. No existe la posición número 20.

¿Cuántos caracteres tiene un string?

La función len() devuelve la cantidad total de caracteres de una cadena.

Python
texto = "Python"
print(len(texto))
Resultado

La función devuelve 6, ya que la palabra "Python" tiene seis caracteres.

Strings inmutables

Un string en Python es inmutable: una vez creado, no se puede cambiar carácter por carácter. Cualquier "modificación" en realidad crea un string nuevo.

Python
texto = "Python"
texto[0] = "J"
print(texto)
¿Por qué falla?

Al ejecutar el bloque anterior obtendrás TypeError: 'str' object does not support item assignment. Los strings no permiten asignar directamente a una posición.

La forma correcta de "cambiar" un carácter es construir un string nuevo, por ejemplo combinando slicing y concatenación:

Python
texto = "Python"
nuevo = "J" + texto[1:]
print(nuevo)
print(texto)
Observa

texto nunca cambió: nuevo es un string completamente distinto. Esto es importante porque significa que los métodos de string (como upper() o replace()) siempre devuelven un resultado nuevo en lugar de modificar el original.

Slicing (Rebanado de cadenas)

El slicing permite extraer una parte de un string sin modificar el original.

Sintaxis

cadena[inicio:fin]

✔ Incluye el índice inicial.
✔ No incluye el índice final.

012345
Python
Python
texto = "Python"
print(texto[0:2])
print(texto[0:4])
print(texto[2:6])

Resultado del slicing

ExpresiónResultadoExplicación
texto[0:2]PyPosiciones 0 y 1
texto[0:4]PythDesde la primera hasta antes del índice 4

Omitiendo índices

Python permite omitir el índice inicial o final.

Python
texto = "Programación"
print(texto[:5])
print(texto[5:])
print(texto[:])

texto[:5]

Desde el inicio hasta antes del índice 5.

texto[5:]

Desde el índice 5 hasta el final.

texto[:]

Devuelve una copia completa del string.

Paso (Step)

El tercer valor indica cuántas posiciones debe avanzar Python.

cadena[inicio:fin:paso]
Python
texto = "Programacion"
print(texto[::2])
print(texto[::3])
print(texto[::-1])
CódigoResultado

texto[::3]Pgmi
texto[::-1]noicamargorP

Métodos de los Strings

Los strings son objetos. Esto significa que poseen métodos, es decir, funciones que permiten transformar o consultar el contenido de una cadena.

Importante

Los métodos de los strings no modifican la cadena original. Devuelven una nueva cadena con el resultado (recuerda la sección Strings inmutables).

upper()

Convierte todos los caracteres a mayúsculas.

Python
texto = "Python"
print(texto.upper())
Uso recomendado

Muy útil para realizar comparaciones de texto ignorando diferencias entre mayúsculas y minúsculas.

lower()

Convierte todos los caracteres a minúsculas.

Python
correo = "Roy.Carrasco@UAB.edu"
print(correo.lower())

capitalize()

Convierte únicamente la primera letra en mayúscula.

Python
texto = "python es divertido"
print(texto.capitalize())

title()

Convierte la primera letra de cada palabra en mayúscula.

Python
nombre = "universidad adventista de bolivia"
print(nombre.title())

swapcase()

Invierte mayúsculas por minúsculas y viceversa.

Python
texto = "PyThOn"
print(texto.swapcase())

casefold()

Es similar a lower(), pero está diseñado para realizar comparaciones de texto de manera más robusta, especialmente con caracteres internacionales.

Python
texto = "PYTHON"
print(texto.casefold())
¿Cuándo usarlo?

Cuando necesites comparar cadenas sin importar mayúsculas, minúsculas o ciertas diferencias de idioma.

MétodoDescripciónEjemplo
upper()Convierte a mayúsculas.PYTHON
lower()Convierte a minúsculas.python
capitalize()Primera letra en mayúscula.Python
title()Cada palabra inicia con mayúscula.Hola Mundo
swapcase()Invierte mayúsculas y minúsculas.pYTHON
casefold()Versión avanzada de lower().python

Alineación y relleno

Estos métodos ajustan el ancho de una cadena rellenando con espacios (o el carácter que indiques).

Python
codigo = "42"
print(codigo.zfill(5))
print("Python".ljust(12, "."))
print("Python".rjust(12, "."))
print("Python".center(12, "*"))
MétodoDescripción
zfill(n)Rellena con ceros a la izquierda hasta completar n caracteres.
ljust(n, c)Alinea a la izquierda y rellena a la derecha con el carácter c.
rjust(n, c)Alinea a la derecha y rellena a la izquierda con el carácter c.
center(n, c)Centra el texto rellenando ambos lados con el carácter c.
Uso recomendado

Muy útiles para alinear columnas en reportes de consola, o para construir códigos con ceros a la izquierda (como 00042).

Comprobaciones is...()

Estos métodos no transforman el string: devuelven True o False según el contenido de la cadena.

Python
print("12345".isdigit())
print("Python".isalpha())
print("Python3".isalnum())
print("   ".isspace())
print("Python 3".isalnum())
MétodoDevuelve True si...
isdigit()Todos los caracteres son dígitos.
isalpha()Todos los caracteres son letras.
isalnum()Todos los caracteres son letras o dígitos (sin espacios ni símbolos).
isspace()Todos los caracteres son espacios en blanco.
Uso recomendado

Muy usados para validar datos ingresados por el usuario, por ejemplo comprobar que una edad ingresada como texto realmente sea un número antes de convertirla con int().

strip()

Elimina los espacios en blanco al inicio y al final de una cadena.

Python
texto = "     Python     "
print(texto.strip())
Muy utilizado

Ideal cuando se reciben datos desde un teclado o desde un archivo de texto.

lstrip() y rstrip()

Permiten eliminar espacios únicamente de un lado.

Python
texto = "     Python     "
print(texto.lstrip())
print(texto.rstrip())
MétodoElimina
lstrip()Espacios de la izquierda
rstrip()Espacios de la derecha

replace()

Reemplaza una parte del texto por otra.

Python
frase = "Me gusta Java"
print(frase.replace("Java","Python"))

split()

Divide una cadena en varias partes utilizando un separador.

Python
cadena = "Juan,Pedro,Ana,Luis"
lista = cadena.split(",")
print(lista)
Importante

El método devuelve una lista, no un string.

join()

Hace exactamente lo contrario que split().

Python
nombres = ["Juan","Pedro","Ana"]
texto = " - ".join(nombres)
print(texto)

find()

Busca una palabra dentro de otra cadena.

Python
texto = "Programación en Python"
print(texto.find("Python"))

Si encuentra el texto devuelve la posición donde comienza.

index()

Funciona parecido a find(), pero genera un error si no encuentra el texto.

Python
texto = "Python"
print(texto.index("t"))
Diferencia

find() devuelve -1 si no encuentra la cadena.

index() produce una excepción.

count()

Cuenta cuántas veces aparece un texto.

Python
texto = "banana"
print(texto.count("a"))

startswith()

Comprueba si una cadena comienza con determinado texto.

Python
archivo = "reporte.pdf"
print(archivo.startswith("rep"))

endswith()

Comprueba cómo termina una cadena.

Python
archivo = "reporte.pdf"
print(archivo.endswith(".pdf"))

f-Strings (Python moderno)

Los f-Strings son la forma recomendada de insertar variables dentro de una cadena.

Sintaxis

Antepone la letra f a la cadena y coloca las variables entre llaves.

Python
nombre = "Roy"
edad = 30
print(f"Hola {nombre}")
print(f"Tienes {edad} años")

Expresiones dentro de un f-String

Dentro de las llaves puedes colocar operaciones y expresiones.

Python
a = 15
b = 7
print(f"La suma es {a+b}")
print(f"El doble es {a*2}")
print(f"La división es {a/b}")

format() y especificadores

Dentro de las llaves de un f-String puedes agregar :especificador para controlar decimales, ancho y alineación.

Python
promedio = 87.4567
print(f"Promedio: {promedio:.2f}")
print(f"{'Nombre':<10}|{'Edad':>5}")
print(f"{'Juan':<10}|{20:>5}")
print(f"{1234567:,}")
EspecificadorEfecto
:.2fRedondea a 2 decimales.
:<10Alinea a la izquierda en un ancho de 10.
:>5Alinea a la derecha en un ancho de 5.
:,Agrega comas como separador de miles.

f-Strings de depuración

Desde Python 3.8, agregar = después de una expresión imprime también su nombre, útil para depurar rápidamente sin escribir print("variable:", variable).

Python
nombre = "Roy"
edad = 30
print(f"{nombre=}")
print(f"{edad=}")
print(f"{edad*2=}")

Errores comunes

❌ Olvidar la letra f

Si escribes "Hola {nombre}" Python mostrará literalmente {nombre}.

❌ Llaves incorrectas

Los f-Strings utilizan { } no paréntesis.

❌ Variables inexistentes

Si la variable no existe se producirá un NameError.

Buenas prácticas

  • Utiliza nombres descriptivos para las variables.
  • Prefiere f-Strings antes que concatenaciones largas.
  • No abuses de las comillas triples para textos pequeños.
  • Utiliza strip() cuando recibas datos del usuario.
  • Convierte el texto con lower() o casefold() antes de comparar.
  • No modifiques directamente una cadena pensando que cambiará el original; recuerda que los strings son inmutables.
